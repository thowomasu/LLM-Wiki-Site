# GitHub Pages Site

This folder is the curated public site for the LLM Wiki research outputs.

Expected project URL after GitHub Pages is enabled:

`https://thowomasu.github.io/LLM-Wiki/`

Recommended GitHub Pages setting:

- Source: Deploy from a branch
- Branch: main
- Folder: `/docs`

Do not publish the whole vault unless you have audited it for raw data, local
paths, unsupported claims, and private material.
