Recommended demo slice

Use cell 59294 at 25 C. Start with DV_WLTP.csv as the variable-current input and EIS_test.csv as the measured impedance target.

This keeps the demo small while still proving the core idea: same cell, same temperature, dynamic discharge and EIS.

Limit

Do not pitch this as a degradation-aging dataset. It is fresh-cell characterization. Good for validating the pipeline mechanics, weak for lifetime or SOH prediction unless paired with an aging dataset.

Paper math check

Wildfeuer, Gieler, and Karger support the same broad DRT direction, but their stronger method jointly fits EIS and time-domain pulse data in one regularized nonnegative inverse problem. The current CALB package has the data needed to attempt that next. The current first slice is still only a dataset and bridge-validation package, not a finished combined-DRT implementation.

Math audit: 00_Docs/DRT_PAPER_MATH_AUDIT.md. Weak spot: our existing time-domain solver compares against EIS after fitting. The paper's combined method fits both domains together.
