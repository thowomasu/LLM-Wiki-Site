Blunt Verdict

The pipeline now runs on Zenodo Expt4 processed GITT, discharge-curve, and hybrid-pulse data. GITT remains the only DRT-like feature track. The other protocols are health-feature evidence. This is not yet equivalent to the DIB validation pipeline because no EIS-derived DRT comparison target was found in the inspected Expt4 folders.

What To Trust

  • Trust the loader, window-finder, and voltage-fit mechanics as a first pass.
  • Trust held-out-cell and trend checks as internal health-feature tests.
  • Treat DRT peak locations as model-sensitive features, not proven physics.
  • Do not claim EIS validation from this package.
