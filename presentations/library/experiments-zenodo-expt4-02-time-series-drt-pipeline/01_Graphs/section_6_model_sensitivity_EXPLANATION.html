What This Figure Shows

This graph tests how the selected GITT fit changes when lambda and baseline mode change.

How To Read It

If RMSE stays low but peak tau moves, the voltage trace is not forcing a unique DRT shape. That is a weak spot, not a plotting issue.

What We Can Learn

The Zenodo GITT fit is model-sensitive enough that peak locations must be treated carefully.

Caveat

Do not tune lambda after looking for a pleasing DRT curve. That would make the result look more certain than it is.
