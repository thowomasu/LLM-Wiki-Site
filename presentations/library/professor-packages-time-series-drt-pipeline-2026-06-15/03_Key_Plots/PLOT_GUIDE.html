The plots are now readable, but the scientific result did not become prettier. That is good. The figures should not hide the problem.

The story is:

1. The pipeline reconstructs voltage well. 2. The DRT-like shape does not match EIS strongly. 3. SOC uncertainty exists. 4. Model assumptions matter. 5. A drive-cycle dataset is needed if the project claim is about drive-cycle behavior.
