What This Figure Shows

This graph compares representative Zenodo Expt4 files before choosing the main pipeline track.

How To Read It

The accepted-window panel tells you which files expose clear pulse/rest candidates. The duration panel tells you how much time-series context each trace contains.

What We Can Learn

Processed GITT is the cleanest first target. Hybrid pulse data may be usable later. Constant-current discharge files are poor DRT candidates unless the goal changes to simple ageing features.

Caveat

This is a screening step, not a fit. It says which files are plausible inputs, not which DRT curves are true.
