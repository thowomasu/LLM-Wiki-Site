Blunt Status

The Zenodo Expt4 track is a working multi-protocol health-feature pipeline. GITT is the only DRT-like feature track. Discharge and hybrid pulse traces are separate health-feature tracks. It is not validated against EIS-derived DRT inside this package. If you present it as validated DRT physics, that is overclaiming.
