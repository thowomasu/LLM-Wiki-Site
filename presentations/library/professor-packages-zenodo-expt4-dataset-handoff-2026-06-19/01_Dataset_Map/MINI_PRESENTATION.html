Mini Presentation: Dataset Map

Zenodo Expt4 Dataset Handoff, professor handoff, 2026-06-19

Slide 1: Results

  • This folder explains what Zenodo Expt4 contains, where it came from, and how the processed files relate.
  • Expt4 is the 0 to 100 percent drive-cycle discharge ageing case.
  • The handoff identifies the Zenodo record, DOI, Expt4 zip name, and published MD5.

Slide 2: Achievements

  • Dataset provenance is documented separately from model results.
  • Column guidance makes the processed files easier to audit.
  • The package avoids copying the full raw dataset into the handoff.
  • The checksum note ties the local source back to the published archive.

Slide 3: Open Issues

  • The full raw dataset is external and large.
  • The map does not prove the extracted features are scientifically valid.
  • Any local rerun still needs the same source archive and extraction structure.

Slide 4: Roadmap

  • Keep DOI, checksum, and source-file identity attached to every future result.
  • Add any professor-confirmed dataset interpretation here first.
  • Record any changes to extraction paths or column assumptions.
  • Use this folder as the first stop when source provenance is questioned.
