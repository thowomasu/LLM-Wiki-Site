Glossary

Plain definitions for acronyms and dataset codes used across the toolkit.

Rule for this site: acronyms should be expanded the first time they appear on a page. This glossary is the backup, not the replacement for clear writing.
Acronym or codeExpansionMeaning in this project
LLMLarge Language ModelThe AI tooling layer used to maintain this research wiki and generate public-facing outputs.
DRTDistribution of Relaxation TimesBattery-analysis method that represents relaxation behavior across time constants.
DIBData in BriefJournal and dataset style used by the Zenodo Experiment 4 workstream.
CALBChina Aviation Lithium BatteryManufacturer name used for the CALB L148N58A prismatic cell dataset.
L148N58A58 Ah large prismatic lithium-ion cell modelThe specific cell model in the CALB paper and Mendeley repository.
EISElectrochemical Impedance SpectroscopyFrequency-domain impedance test used as the comparison target for DRT outputs.
TDMTime-Domain MeasurementTime-series voltage/current data used to fit relaxation behavior from pulses or dynamic tests.
HPPCHybrid Pulse Power CharacterizationPulse protocol used to extract time-domain relaxation windows.
SOCState of ChargeBattery charge level. Bad SOC alignment can make comparisons unfair.
SOHState of HealthBattery ageing or degradation state. The current CALB work does not validate SOH prediction.
OCVOpen Circuit VoltageRested voltage used to match pulse candidates with EIS records.
ECMEquivalent Circuit ModelCompact battery model used as a baseline in drive-cycle audits.
CALCECenter for Advanced Life Cycle EngineeringExternal battery-data source used for candidate validation planning.
NASANational Aeronautics and Space AdministrationExternal battery dataset source considered after CALCE and corrected-EIS RADAR4KIT.
PLNCALCE cell identifierIdentifier used in CALCE storage files to join capacity, SOC, temperature, storage period, and impedance records.
UDDSUrban Dynamometer Driving ScheduleDynamic driving profile used in CALB drive-cycle tests.
US06High-aggression supplemental driving scheduleDynamic driving profile used in CALB drive-cycle tests.
WLTPWorldwide Harmonised Light Vehicles Test ProcedureDynamic driving profile used in CALB drive-cycle tests.
RPTReference Performance TestPeriodic characterization test. In CALB, HPPC and C/20 tests are RPT-like but not literally named RPT.
