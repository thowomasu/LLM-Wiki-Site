CALB Web Manual

How to use the website, data files, and pipeline without guessing.

Use the website

  1. Open the presentation and read the first claim boundary.
  2. Use the left navigation to jump to Data, Exploration, Drive cycles, Limits, and Glossary.
  3. Open the data explorer when a section references a CSV or JSON file.
  4. Use the source links to compare local outputs with the paper and Mendeley repository.

Open presentation Open data explorer

Use the pipeline

Run scripts from the repository root with the DRT conda interpreter. The main guardrail is the regression check. Do not change claims until it still passes.

& "E:\anaconda\envs\DRT\python.exe" -B scripts\experiments\calb_l148n58a\calb_l148n58a_regression_checks.py

Glossary Reminder

EIS means Electrochemical Impedance Spectroscopy. HPPC means Hybrid Pulse Power Characterization. DRT means Distribution of Relaxation Times. SOC means State of Charge. OCV means Open Circuit Voltage. ECM means Equivalent Circuit Model. PLN is a CALCE external-dataset cell identifier, not a CALB cell ID.
