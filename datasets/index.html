Every dataset currently used or screened by the LLM Toolkit work.

Purpose: this page lists every dataset currently used or explicitly screened by the public project outputs. It separates current evidence from future validation targets.

CALB L148N58A Dataset

China Aviation Lithium Battery L148N58A large-prismatic-cell dataset used for the current EIS (Electrochemical Impedance Spectroscopy), HPPC (Hybrid Pulse Power Characterization), and drive-cycle evidence package.

246 / 246
Internal pulse-bridge quality-pass rows. This is the strongest current local result.

Zenodo Experiment 4 DIB Dataset

Data in Brief style ageing dataset used for the time-series DRT pipeline, feature extraction, held-out-cell checks, and negative-control audits.

1,106
Local files inspected for same-cell EIS evidence. The result was zero EIS-like hits, so this stays a health-feature dataset, not EIS validation.

CALCE Storage 25 C Dataset

Center for Advanced Life Cycle Engineering storage dataset screened as the first external EIS validation candidate.

24
Same-PLN candidate rows at 25 C. PLN means the CALCE cell identifier used to join capacity and impedance records.

NASA Battery Dataset

National Aeronautics and Space Administration battery dataset screened as a later validation candidate.

34
Unique cells with charge, discharge, and impedance evidence. Useful later, but weaker than CALCE for the next step.

KIT RADAR4KIT Dataset

Comprehensive battery ageing dataset with capacity and impedance fade measurements. Selected as the better second-dataset candidate after the Zenodo Experiment 4 audit.

Local DOI 10.35097/1947 cell_eisv2.zip is incomplete; use corrected EIS addendum before validation.
