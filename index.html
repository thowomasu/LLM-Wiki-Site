What This Project Is

This is the front-facing part of the LLM (Large Language Model) toolkit wiki. It collects the work done so far on battery-data pipelines, DRT (Distribution of Relaxation Times) modeling, dataset qualification, professor-facing reports, source traces, and downloadable evidence.

The goal is simple: a new reader should be able to open this site, understand what was built, inspect the actual data and plots, and see the limits without digging through the private vault.

Claim boundary: this site is for clear review, not hype. It does not turn internal results into external validation. The strongest CALB (China Aviation Lithium Battery) result is internal pulse-bridge evidence, and the clean next step is a frozen-rule CALCE (Center for Advanced Life Cycle Engineering) run with an ECM (Equivalent Circuit Model) baseline.

What Has Been Done So Far

DRT (Distribution of Relaxation Times) pipeline

Built a time-domain battery pipeline that finds pulse/rest windows, fits DRT-like relaxation curves, compares against EIS (Electrochemical Impedance Spectroscopy), and reports where voltage fitting can mislead.

Open DRT presentations

DIB (Data in Brief) dataset work

Mapped and tested the Zenodo Experiment 4 / DIB-style dataset package, including data audit, feature extraction, held-out-cell checks, and reproducibility material.

Open DIB presentations

CALB dataset work

Parsed the CALB L148N58A archive, published viewable data files, built the pulse-bridge evidence package, and kept the drive-cycle limitation visible.
